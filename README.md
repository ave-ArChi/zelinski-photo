# Zelinski Photo — Full Clean (soft borders)

- Прежний дизайн: центрированный крупный заголовок, левое меню.
- Мягкие разделители: классы `border-soft` на верхней/левой/нижней границах.
- Кнопка Telegram в меню и ссылка в футере.
- Галерея `/portfolio` — masonry без обрезки, читает `public/images`.
- Страницы `/about`, `/pricing` статичные.
- Фото в архив не включены.

## Запуск
```bash
npm install
npm run dev
```
Открой `http://localhost:3000` и добавь свои изображения в `public/images`.
